# Paul's Site

This SvelteKit project is a bio/blog fusion personal site. It's easy to set up yourself.

## Developing

Once you've downloaded/cloned this project and installed dependencies with `pnpm install`, start a development server:

```bash
npm run dev
```

## Vercel

This project is hosted by Vercel, and deployments are automated via the Vercel deployment pipeline. You can see CI/CD messages and checks in the PR history.
